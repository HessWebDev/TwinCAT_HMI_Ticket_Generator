This is a project for creating tickets through TwinCAT HMI.

Users will be able to use this HMI to do the following..

1. Add a title (Customer name) 
2. contact information 
  a. Customer Name 
  b. Company Name 
  c. Email 
  d. Phone Number 
4. Notes (Description of what happened)
5. Attachments 
  a. Images 
  b. Code Snipets 
  c. Projects 


Once user fills out the above information the program will do the following..


1. Create a Folder Named as the customer Name 
2. Create a sub txt file with the description 
3. add any images as well as program to the folder as well 
4. Create a notification for end of day to remind you to create a ticket for the above information.
